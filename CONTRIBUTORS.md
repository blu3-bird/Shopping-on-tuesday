## ğŸ› ï¸ Contribution Workflow

Follow these steps to contribute:

1. **Fork** the repository
2. **Clone** your fork locally:

   ```bash
   git clone https://github.com/your-username/ml-for-dummies.git
   cd ml-for-dummies
   ```
3. **Create a new branch** for your work:

   ```bash
   git checkout -b docs/your-change-name
   ```
4. **Make your changes** (documentation, notebooks, or templates)
5. **Commit your changes** with a clear message:

   ```bash
   git commit -m "docs: improve contribution guidelines"
   ```
6. **Push** your branch to GitHub:

   ```bash
   git push origin docs/your-change-name
   ```
7. **Open a Pull Request** on GitHub
